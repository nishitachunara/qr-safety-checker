## 📌 About This Project

QR codes are everywhere — restaurants, shops, posters, and more. But **not all QR codes are safe**. Hackers use fake QR codes to redirect users to phishing sites that steal personal information.

This tool helps users **analyze a URL** from a QR code before visiting it.

---

## 🔍 What It Checks

| Check | Description |
|-------|-------------|
| 🔒 HTTPS | Is the connection secure? |
| 🚩 Suspicious Words | Does it contain words like "login", "verify", "bank"? |
| 🌐 IP Address | Is it using an IP instead of a real domain? |
| 📏 URL Length | Is the URL unusually long? |
| 🔗 URL Shorteners | Is the real destination hidden? |
| 🔤 Fake Characters | Does it use lookalike letters to trick you? |
| @ Symbol | Does it contain @ to redirect elsewhere? |
